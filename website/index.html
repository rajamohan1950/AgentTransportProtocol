Why ATP?

MCP discovers tools. RAG retrieves chunks. ATP orchestrates entire agent economies with trust, routing, compression, and fault tolerance built in.

MCP

Model Context Protocol
  • ● Tool discovery & invocation
  • ✗ No agent identity
  • ✗ No trust scoring
  • ✗ No multi-agent routing
  • ✗ No context compression
  • ✗ No fault tolerance
  • ✗ No economic optimization
  • ✗ No QoS guarantees

Five-Layer Architecture

Inspired by TCP/IP. Each layer solves one problem perfectly. Stack them together for unstoppable multi-agent networking.

L1

Identity & Trust

Ed25519 DIDs, time-decayed trust scoring, Sybil resistance

Trust Formula

T(a) = Σ(q_i × e^(-0.01×Δdays) × γ(task))
       / Σ(e^(-0.01×Δdays) × γ(task))

γ(Analysis)       = 1.2
γ(CreativeWriting) = 1.0
γ(DataProcessing)  = 0.8

L2

Capability Handshake

3-phase SYN/SYN-ACK/ACK negotiation with QoS contracts

▼

Protocol Flow

Phase 1: CAPABILITY_PROBE
  Requester → broadcast task + QoS

Phase 2: CAPABILITY_OFFER
  Agents → scored bids (quality, cost)

Phase 3: CONTRACT_ACCEPT
  Requester → binding agreement

State Machine:
  Idle → ProbeSent → OffersReceived
    → Contracted | Failed

L3

Semantic Context Differentials

28x context compression via cosine similarity & MSC extraction

▼

SCD Pipeline

1. Task Embedding
   task → dense vector in R^d

2. Cosine Similarity
   cos(e_task, e_chunk) = (a·b)/(|a||b|)

3. Minimal Sufficient Context
   MSC = {chunk : cos > threshold}

4. Context Diff
   50,000B → 1,768B (28.3x)

5. Adaptive Refinement
   confidence < 0.7 → CONTEXT_REQUEST

API Documentation

Two flavors: verb functions (print) and noun functions (return values). Every type implements Display.

benchmark() verb

pub fn benchmark()
// Prints the full 7-scenario benchmark table. Zero setup.
Runs 50 agents, 10,000 tasks, all 7 scenarios (including ablation). Prints a formatted table to stdout. One line, done.

bench(tasks) noun

pub fn bench(tasks: usize) -> BenchReport
// Returns a BenchReport with all metrics.
Returns a BenchReport with methods: .atp(), .baseline(), .scenario("name"), .all(). Implements Display for pretty printing.

route(skill) / find_route(skill) verb noun

pub fn route(skill: &str)                             // prints
pub fn find_route(skill: &str) -> RouteResult         // returns
pub fn find_route_with(skill: &str, min_quality: f64) -> RouteResult
Skills: "coding", "analysis", "writing", "data" (case-insensitive, many aliases). Returns RouteResult with .quality, .cost, .latency_ms, .pattern, .agents.

compress(data, skill) / shrink(data, skill) verb noun

pub fn compress(data: &[u8], skill: &str)                // prints
pub fn shrink(data: &[u8], skill: &str) -> CompressResult   // returns
Returns CompressResult with .ratio, .original_size, .compressed_size, .chunks, .confidence.

agent() / sign(msg) verb noun

pub fn agent() -> Agent                      // new identity
pub fn sign(message: &[u8])                   // prints agent + sig + verify

// Agent methods:
agent.did() -> &str                          // "did:key:z6Mk..."
agent.sign(msg) -> Signature
agent.verify(msg, &sig) -> bool
Generates a fresh Ed25519 keypair with W3C DID. Sign any bytes, verify signatures. All types implement Display.

trust(skill) / trust_score(skill) verb noun

pub fn trust(skill: &str)                         // prints
pub fn trust_score(skill: &str) -> TrustInfo       // returns
Returns TrustInfo with .score (0.0–1.0) and .samples (interaction count).

Network type

pub struct Network { /* ... */ }

Network::new()                              // 50 agents, seed 42
Network::with_seed(99)                       // custom seed
net.agents() -> usize
net.route("coding") -> RouteResult
net.benchmark(10_000) -> BenchReport
net.compress(data, "coding") -> CompressResult
net.trust("coding") -> TrustInfo
For advanced use only. Free functions use a global Network automatically. Create your own for custom seed or agent count.

Granular API (Advanced)

import atp

# Identity
kp = atp.KeyPair()
print(kp.did_uri())           # "did:key:z6Mk..."
sig = kp.sign(b"hello")
assert kp.verify(b"hello", sig)

# Routing
router = atp.EconomicRouter()
cap = atp.Capability(atp.TaskType.code_generation(), 0.9, 100, 0.05)
router.add_agent(atp.AgentId(), [cap], 0.85)
route = router.find_route(atp.TaskType.code_generation(), atp.QoSConstraints())

# Simulation
sim = atp.Simulation(agents=50, seed=42)
results = sim.run_benchmark(tasks=10000)
for r in results:
    print(r)

Get Started in 30 Seconds

Add ATP to your project and start routing tasks across agent networks.

Rust

# Cargo.toml
[dependencies]
atp-sdk = { git = "https://github.com/AForge-AI-Labs/agent-transport-protocol" }
// main.rs — That's it. Seriously.
fn main() {
    atp_sdk::benchmark();
    atp_sdk::route("coding");
    atp_sdk::compress(b"context...", "coding");
}

Run: cargo run
